Motivar - Frases Motivacionais

Um aplicativo móvel incrível criado por Jesiel Gomes, utilizando React Native e uma API RESTful. Disponível na Play Store.

Detalhes do Projeto

Este aplicativo móvel foi desenvolvido com as mais recentes tecnologias, incluindo React Native e uma API RESTful, garantindo uma experiência do usuário excepcional. Abaixo estão alguns dos destaques:

  • Design responsivo para uma ótima visualização em dispositivos móveis.
  • Integração de animações modernas para uma experiência interativa.
  • Uso de React Native para eficiência no desenvolvimento cross-platform.
  • Implementação de práticas recomendadas de acessibilidade.

Explore o Aplicativo

Em busca de inspiração diária? O aplicativo "Frases Motivacionais Diárias" é a sua dose de positividade no bolso! Seja a sua fonte de motivação, ânimo e força todas as manhãs com nossa coleção cuidadosamente selecionada de frases motivacionais.

Funcionalidades

  • Salvar nos favoritos
  • Compartilhar com outras pessoas
  • Selecionar autor.
  • Dark Mode.
  • Visualizar

Conclusão e Próximos Passos
